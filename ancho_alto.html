Width (Ancho) Height (Alto) Las propiedades height y width no incluyen relleno, bordes ni márgenes. Establece la altura/anchura del área dentro del relleno, el borde y el margen de el elemento. VALORES Las propiedades pueden tener los siguientes valores: { auto } - Este es el valor predeterminado, el navegador calcula la altura y la anchura { length } - Define la altura/anchura en px, cm, etc. { % } - Define la altura/anchura en porcentaje del bloque contenedor { initial } - Establece la altura/anchura en su valor predeterminado { inherit } - La altura/anchura será heredado de su valor primario
